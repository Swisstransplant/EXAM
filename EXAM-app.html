What is EXAM?

EXAM (ex vivo allograft monitoring) is an analytics dashboard for hypothermic machine perfusion data in deceased-donor kidney transplantation. It supports data from the ORS LifePort Kidney Transporter.

How was KIDMO developed?

EXAM was developed by Swisstransplant—the Swiss National Foundation for organ donation and transplantation and the transplant centers. Further information can be found in the reference below.

Schwab S, Steck H, Binet I, Elmer A, Ender W, Franscini N, Haidar F, Kuhn C, Sidler D, Storni F, Krügel N, Immer F. EXAM: Ex vivo allograft monitoring dashboard for the analysis of hypothermic machine perfusion data in deceased-donor kidney transplantation. Research Square. 2023. doi:10.21203/rs.3.rs-2713168/v1

Contact

For inquiries about EXAM please contact Simon Schwab simon.schwab@swisstransplant.org

Important

Hypothermic machine perfusion data alone should never be used as the sole basis to discard a kidney allograft.
